# P2P-system

## How to run
1. Go to /jar directory
* Client.jar is in /Client directory
* Server.jar is in /Server directory

2. Run Server:
`$java -jar Server.jar`

3. Run Client1:
`$java -jar Client.jar`

4. Run Client2:
`$java -jar Client.jar`

Please refer Github README.md for more instruction details
https://github.com/aranne/P2P-system
